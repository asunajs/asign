## 配置

在 `index.js` 旁边新建文件，例如叫 `asign.config.mjs`（然后根据本文裆进行配置）

## 核心代码

`index.js`：以 `ecloud` 为例，因为华为云函数的特殊性，一定手动指定配置文件路径，如下所示。

```js
const { run } = require('@asunajs/ecloud');
const path = require('path');

exports.handler = async (event, context) => {
  return await run(path.resolve(__dirname, './asign.config.mjs'));
};
```
